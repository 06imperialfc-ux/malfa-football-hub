# MALFA Live Launch Checklist

## Database

- [ ] Supabase project created
- [ ] Strong database password stored safely
- [ ] `supabase-schema.sql` run successfully
- [ ] First administrator created
- [ ] Administrator added to `public.admin_users`
- [ ] Public sign-ups disabled

## Website connection

- [ ] Project URL added to `js/cms-config.js`
- [ ] Publishable key added to `js/cms-config.js`
- [ ] No secret/service-role key committed
- [ ] Admin page displays **Live CMS**
- [ ] Administrator login tested

## Content test

- [ ] Test club and crest uploaded
- [ ] Test fixture published
- [ ] Test result verified
- [ ] Table rebuilt and ranked correctly
- [ ] Test news article published
- [ ] Test partner added
- [ ] Hidden tournament tested

## Deployment

- [ ] GitHub repository created
- [ ] Website files uploaded to repository root
- [ ] GitHub Pages enabled from main branch/root
- [ ] Live public URL tested on desktop and mobile
- [ ] Admin page tested from live URL
- [ ] Temporary test content removed or clearly marked

## Presentation readiness

- [ ] Association name and logo approved
- [ ] Contact email confirmed
- [ ] At least three real club profiles added
- [ ] At least one division table demonstrated
- [ ] At least one fixture and verified result demonstrated
- [ ] Sponsors and partners page reviewed
- [ ] President presentation link and backup screenshots prepared
